# Quiet Lens — UI Kit

Menu bar app for macOS. Dims and blurs every window except the one you're working in.

## What's here
- `MenuBar.jsx` — the macOS menu bar with the Lens status item
- `LensPanel.jsx` — the dropdown that opens from the status item
- `Desktop.jsx` — the simulated desktop (focused vs. dimmed windows) so you can see Lens in context
- `index.html` — interactive demo: toggle the panel, switch modes, see the scrim apply

## Status
**Forward-looking mock.** The real app's SwiftUI source wasn't in hand; this is the existing concept rebuilt with the new design system.
