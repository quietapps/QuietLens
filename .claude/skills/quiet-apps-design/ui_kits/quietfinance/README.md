# Quiet Finance — UI Kit

Native macOS net-worth tracker. SwiftUI + SwiftData. 100% local.

## What's here
- `Window.jsx` — macOS window chrome (traffic lights, toolbar, sidebar)
- `Sidebar.jsx` — account groups + currency switcher
- `AccountRow.jsx` — one row in the accounts list
- `NetWorthHero.jsx` — the big number + delta + sparkline + snapshot chart
- `SnapshotSheet.jsx` — modal sheet for adding a new snapshot
- `index.html` — interactive: switch accounts, add a snapshot, see the chart move

## Status
**Forward-looking mock.** The real app is SwiftUI; this is the existing concept rebuilt with the new design system.
