# Quiet Notch — UI Kit

Small ambient widget that lives inside the MacBook Pro notch. Hover or click to expand into a panel.

## What's here
- `MacbookFrame.jsx` — top of a MacBook lid (the notch is real geometry)
- `Notch.jsx` — the collapsed pill + expanded panel with three modes (Now Playing, Focus Timer, AirDrop)
- `index.html` — interactive: hover the notch to expand, click chips to switch modes

## Status
**Forward-looking mock.** The Swift app exists; this kit is the existing concept rebuilt with the new design system.
